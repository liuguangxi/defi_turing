# Défi Turing Unofficial Offline Edition in Typst

Collect all problems from [Défi Turing](https://www.apprendre-en-ligne.net/turing). The content has been translated into English.

To compile the document, you need to install [`typst`](https://github.com/typst/typst) first, then run

```bash
typst compile defi_turing_offline.typ defi_turing_offline.pdf
```

to get a dynamically updated PDF file [defi_turing_offline.pdf](./defi_turing_offline.pdf).
